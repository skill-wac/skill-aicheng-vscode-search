# skill&爱成 - 对话搜索

> Skill（AI）和爱成兄共同打造的 Claude Code 对话搜索工具。

## 做什么的？

使用 Claude Code 久了，对话记录散落在几十个会话里，想回顾之前聊过的话题？翻半天找不到？

**skill&爱成** 一键搜索所有历史对话，关键词匹配，秒出结果。

## 怎么用？

- `Ctrl+Alt+F` — 弹出搜索面板
- 点击 VS Code 右下角 **🔍 skill&爱成**
- `Ctrl+Shift+P` → 输入"搜索历史对话"

## 安装

1. 下载 `skill-aicheng-search-1.0.0.vsix`
2. VS Code 中 `Ctrl+Shift+P` → "Extensions: Install from VSIX"
3. 选择下载的 `.vsix` 文件
