# skill&爱成 - 对话搜索

> 🔍 搜索 Claude Code 历史对话 — 由 **Skill（AI）** 和 **爱成兄** 共同打造

[![VS Code](https://img.shields.io/badge/VS%20Code-1.80%2B-blue)](https://code.visualstudio.com/)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

## 为什么需要？

Claude Code 对话长了，想回顾之前聊过的话题？翻几十页聊天记录？

**skill&爱成** — 一键搜索所有历史对话，弹出面板，秒出结果。

## 使用方式

| 触发 | 操作 |
|------|------|
| ⌨️ 快捷键 | `Ctrl+Alt+F` |
| 🖱️ 状态栏 | 点击右下角 **🔍 skill&爱成** |
| 🎯 命令面板 | `Ctrl+Shift+P` → "搜索历史对话" |

## 安装

1. 下载 [skill-aicheng-search-1.0.0.vsix](https://github.com/skill-wac/skill-aicheng-vscode-search/releases/latest)
2. VS Code → `Ctrl+Shift+P` → "Extensions: Install from VSIX"
3. 选择 `.vsix` 文件，完成

## 功能

- 🔎 搜索 79+ 个 Claude Code 会话记录
- 🏷️ 快捷标签一键搜索
- 📋 点击结果复制文本
- 🎨 暗色主题 UI，匹配 VS Code
